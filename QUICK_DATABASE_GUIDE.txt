WHY USE DATABASE?
═════════════════

❌ OLD WAY (CSV Loading):
   • Takes 30-60 seconds
   • Limited to 100 stocks
   • Slow sorting/filtering
   • Repeated API calls

✅ NEW WAY (Database Cache):
   • Takes < 1 second (60x faster!)
   • ALL stocks supported
   • Instant sorting/filtering
   • One-time setup, then blazing fast forever


SUPER SIMPLE 3-STEP SETUP
══════════════════════════

Step 1: Open Dashboard
──────────────────────
http://localhost:5050
Press Ctrl+Shift+R (hard refresh)

Step 2: Go to "All Stocks" Tab
──────────────────────────────
Click the "All Stocks" tab

Step 3: One-Time Database Sync
───────────────────────────────
Click "🔄 Sync Database First" button
Wait ~2 minutes (one-time only!)
Click "⚡ Load from Database" button
DONE! Data loads instantly now!


DAILY WORKFLOW (2 BUTTONS!)
════════════════════════════

Every trading day, just:

1. Click "📅 Update Today Only" (~10 sec)
   └─ Fetches only today's new prices

2. Click "⚡ Load from Database" (instant!)
   └─ See all data with today's updates

That's it! No more waiting!


BUTTONS EXPLAINED
═════════════════

🟢 Sync Database First
   ├─ Use: First time only (or when you add new CSVs)
   ├─ Time: ~2 minutes
   └─ Does: Loads all CSV files into database

🔵 Load from Database
   ├─ Use: EVERY TIME you want to view stocks
   ├─ Time: < 1 second
   └─ Does: Shows all stocks instantly from database

🟠 Update Today Only
   ├─ Use: Daily, after market close
   ├─ Time: ~10 seconds
   └─ Does: Updates database with today's prices

⚫ DB Stats
   ├─ Use: Anytime (to check database)
   ├─ Time: Instant
   └─ Does: Shows database info (records, size, etc.)

🔴 Load from CSV (Slow)
   ├─ Use: ONLY if database not synced
   ├─ Time: 30-60 seconds
   └─ Does: Old slow method (avoid this!)

TROUBLESHOOTING
═══════════════

Problem: "Database not initialized" error
Solution: Click "Sync Database First" button

Problem: Nothing loads
Solution: 
  1. Hard refresh (Ctrl+Shift+R)
  2. Check backend is running: http://localhost:5050/api/db/stats
  3. If backend stopped, restart: ./run_dashboard.sh

Problem: Want to rebuild database
Solution:
  1. Delete: rm stock_cache.db
  2. Click "Sync Database First" again
